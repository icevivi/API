##  如何使用源码编译 Qt 

基于 LGPL 协议，如果您需要自己使用Qt源码进行编译，可以参考本节内容。

需要注意的是，如果使用不同的Qt版本，有可能会产生不兼容造成程序不能正常运行的情况。请确定您选择了正确的Qt版本，并且没有破坏其接口。

我们不同的产品会使用不同的 Qt 模块，所以编译时请注意需要包含这些模块，如果缺少部分模块，会影响程序运行。

### Windows平台

1、下载对应版本的 .zip 文件，并解压；
2、安装 Visual Studio 2015 (VC++)； 
3、安装 Qt 编译依赖的其它库，这个需要按您所需要使用的模块和选项来选择安装；
4、打开 Visual Studio 命令行工具，在解压后的 Qt 源码目录下，运行 configure ，按您的需要设置命令行参数；
5、上一步执行成功后，运行 nmake ；
6、上一步执行成功后，运行 nmake install。

### Linux平台

1、下载对应版本的 .tar.xz 文件，并解压；
2、安装 Qt 编译依赖的包，这个需要按您所需要使用的模块和选项来选择安装；
3、在解压后的 Qt 源码目录下运行 ./configure ，并按您的需要设置命令行参数；
4、上一步执行成功后，运行 make ；
5、上一步执行成功后，运行 make install。
